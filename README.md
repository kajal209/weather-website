# Weather Forecast App

This is my Weather Forecast project built as a learning exercise in web development.

## Features
- Displays current weather for any city
- Shows temperature, humidity, and weather conditions
- Responsive design for mobile and desktop

## Technologies
- HTML
- CSS
- JavaScript
- Optional APIs (like OpenWeatherMap)

## How to Use
- Open `index.html` in a browser
- Enter a city name to see the weather
